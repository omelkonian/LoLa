To run:
```bash
> python dyck.py -h
usage: dyck.py [-h] [-n [N]] [-w [W]] [-g [G]] [--rules]

Check your D3 grammar.

optional arguments:
  -h, --help  show this help message and exit
  -n [N]      number of "abc" occurences
  -w [W]      single word to check
  -g [G]      grammar to use
  --rules     print all rules
```
